# Hiërarchisch Accordeon Web App

Dit is een eenvoudige webapplicatie waarmee gebruikers een dynamische, hiërarchische accordeonstructuur kunnen bouwen.

## Functies

- Voeg root-level accordeon nodes toe.
- Voeg geneste (kind) nodes toe aan elke node.
- Bewerk de titel van elke node.
- Verwijder nodes (inclusief alle kinderen).
- Klap nodes in en uit door op de titel of de `+/-` knop te klikken.

## Hoe te gebruiken

1. Open `index.html` in een webbrowser.
2. Klik op de knop "Voeg Hoofd Node Toe" om te beginnen.
3. Gebruik de knoppen binnen elke node om kinderen toe te voegen, de titel te bewerken of de node te verwijderen.
